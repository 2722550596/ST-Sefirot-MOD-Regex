> ## 使用方法：
> 

### 步骤1.
用共创的正则替换原卡的`正文-诡秘之主`局部正则，导入后新正则默认排在最下面，要长按正则最左侧由小点组成的 拖动柄，拖动放在第二个，即`开局-诡秘之主`和`隐藏思维链（thinking）`之间

### 步骤2：
用新的序列基准总值（在这层→https://discord.com/channels/1456496749022806193/1476975772987166814/1477196476592689223）替换原卡的`1源堡`世界书中的`【数值】非凡途径基准上限，装备，扮演法数值参考（强调）`条目中`高序列`的部分

### 步骤3：
把以下内容替换原卡的`1源堡`世界书中的`【判定】判定+DC`条目中的判定DC基准总值部分

`序列 0（真神）：基准总值 1200000

序列 旧日：基准总值 2000000

序列 支柱：基准总值 5000000`

### 步骤4：
在“1源堡”世界书中，新增条目`[快捷指令]`，无需开启。搞好后应该是像这张图一样的→https://discord.com/channels/1456496749022806193/1476975772987166814/1477677119231365164
